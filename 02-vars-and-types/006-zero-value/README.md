# Zero value
- understanding zero value
  - false for booleans
  - 0 for integers
  - 0.0 for floats
  - "" for strings
  - nil for 
    - pointers
    - functions
    - interfaces
    - slices
    - channels
    - maps
- use short declaration operator as much as possible
- use var for
  - zero value
  - package scope
